# 底層重構

## Life is a legacy system that requires constant refactoring.

> 人生和軟件系統其實很像。
> 都會積累技術債，都會經歷失控，也都需要不斷重構。

《底層重構》是一部持續連載的，真實人生回憶錄，寫的是一個被三百塊錢趕出學校的少年，後來如何一步一步把自己的人生重新搭起來。

這裏沒有成功學，也沒有技術雞湯。這裏只有輟學、放鴨、抓鱔魚、學手藝、進城、寫代碼，和一個從湖北洪湖農村走出來的人，如何在一次次失控之後，把自己從舊系統裏慢慢重構出來的真實過程。

### 為什麼寫《底層重構》

人生和軟件系統其實很像。都會積累技術債，都會經歷失控，也都需要不斷重構。

把這些經歷寫下來，既是對自己的交代，也是想把那些走過的彎路、吃過的苦、重新站起來的過程，留給後來的人看一看。
